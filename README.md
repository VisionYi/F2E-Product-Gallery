## 專案使用的技術
- CSS Grid
- SCSS
- Pracel 零配置極速打包工具

## 開發專案使用指令
```shell
# 初始化專案
npm install

# 開啟專案 server
npm run start

# 建置發布的資料 dist
npm run build

# 自動佈署到 Github Page 上
npm run deploy
```
